# Candlebot扩展更新功能说明

## 修复和新增功能

### 1. 自动截图功能（主要更新）
- **点击扩展图标自动截图**：在支持的图表页面上，点击扩展图标会自动截图并跳转到分析页面
- **智能检测**：自动检测当前页面是否为支持的图表页面
- **失败处理**：如果自动截图失败，会显示popup让用户手动操作

### 2. 改进的数据传输
- **优先使用内容脚本**：通过内容脚本传递截图数据，避免URL参数长度限制
- **备用方案**：如果内容脚本失败，小数据使用URL参数，大数据显示错误提示
- **可靠的数据存储**：使用sessionStorage临时存储截图数据

### 3. 优化的popup界面
- **重新截图按钮**：在popup中添加"重新截图"按钮
- **智能按钮显示**：根据是否有截图数据动态显示按钮
- **状态反馈**：显示详细的连接状态和页面检测结果

## 使用流程

### 自动截图流程（推荐）
1. 访问支持的图表网站（如TradingView、aggr.trade等）
2. 打开具体的图表页面
3. 点击浏览器工具栏中的Candlebot扩展图标
4. 扩展会自动：
   - 检测页面类型
   - 截图当前图表
   - 提取图表信息（交易对、交易所、时间框架等）
   - 跳转到Candlebot分析页面
   - 自动填充截图数据

### 手动操作流程
1. 如果自动截图失败，会显示popup
2. 在popup中点击"截图并分析"按钮
3. 或者点击"重新截图"按钮重新尝试

## 故障排除

### 常见问题

1. **截图失败**
   - 确保页面完全加载
   - 检查扩展权限
   - 尝试重新加载页面

2. **数据传输失败**
   - 检查Candlebot web app是否运行
   - 检查环境设置（开发/生产）
   - 尝试重新截图

3. **页面检测错误**
   - 确保访问的是具体的图表页面，不是首页
   - 检查URL是否符合支持的格式

### 调试方法

1. 打开Chrome开发者工具（F12）
2. 切换到"控制台"标签页
3. 查看扩展输出的日志信息
4. 检查是否有错误信息
