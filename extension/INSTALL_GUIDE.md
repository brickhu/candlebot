# Candlebot扩展安装和测试指南

## 快速开始

### 1. 安装扩展

#### 方法A：从源码加载（开发模式）

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd candlebot/extension
   ```

2. **生成图标**（如果还没有PNG图标）
   ```bash
   # 如果有ImageMagick
   brew install imagemagick
   ./create-simple-icons.sh

   # 或者手动创建简单的图标
   echo "创建基本图标..."
   convert -size 16x16 xc:#667eea icons/icon16.png
   convert -size 48x48 xc:#667eea icons/icon48.png
   convert -size 128x128 gradient:#667eea-#764ba2 icons/icon128.png
   ```

3. **在Chrome中加载扩展**
   - 打开Chrome浏览器
   - 进入 `chrome://extensions/`
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `candlebot/extension` 目录

4. **验证安装**
   - 扩展图标应出现在浏览器工具栏
   - 点击图标应显示Candlebot弹窗
   - 检查控制台是否有错误

#### 方法B：打包安装（生产模式）

1. **打包扩展**
   ```bash
   # 在extension目录下
   zip -r candlebot-extension.zip . -x "*.git*" "node_modules/*" "*.md" "test-*"
   ```

2. **安装打包文件**
   - 打开 `chrome://extensions/`
   - 开启开发者模式
   - 将zip文件拖放到页面中

### 2. 配置环境

#### 设置Web应用URL

扩展支持两种环境：

1. **本地开发环境**（默认）
   - Web应用URL: `http://localhost:5173`
   - 用于开发和测试

2. **生产环境**
   - Web应用URL: `https://chat.candlebot.app`
   - 用于实际使用

**切换环境方法：**
- 点击扩展图标打开弹窗
- 在"设置"区域选择环境
- 点击"测试连接"验证

#### 验证连接

1. **确保Web应用运行**
   ```bash
   # 在app目录下
   cd ../app
   npm run dev
   # 或
   bun run dev
   ```

2. **测试扩展连接**
   - 打开扩展弹窗
   - 点击"测试连接"按钮
   - 应显示"已连接"状态

### 3. 基本测试

#### 测试1：网站识别

访问以下网站，观察扩展图标状态：

| 网站 | 预期行为 |
|------|----------|
| `https://www.tradingview.com/chart/BTC/USD` | 图标高亮，显示"点击截图并分析" |
| `https://www.tradingview.com/` | 图标正常，显示"请打开具体图表页面" |
| `https://aggr.trade/SOL/USDT` | 图标高亮，显示"点击截图并分析" |
| `https://www.google.com/` | 图标灰色，显示"请在支持的图表网站使用" |

**验证方法：**
- 鼠标悬停在扩展图标上查看提示
- 点击图标查看弹窗内容

#### 测试2：截图功能

在支持的图表页面：

1. **自动模式**
   - 直接点击扩展图标（不打开弹窗）
   - 应自动截图并跳转到分析页面

2. **手动模式**
   - 点击扩展图标打开弹窗
   - 点击"截图并分析"按钮
   - 应截图并跳转到分析页面

3. **仅截图**
   - 在弹窗中点击"仅截图"按钮
   - 应截图并在弹窗中显示预览

**预期结果：**
- 截图成功，显示预览
- 自动跳转到 `http://localhost:5173/new`（本地环境）
- URL包含 `?from_extension=true` 参数

#### 测试3：数据传递

验证截图数据正确传递到Web应用：

1. **检查URL参数**
   ```
   http://localhost:5173/new?from_extension=true&site=TradingView&symbol=BTCUSD
   ```

2. **检查页面数据**
   - 打开浏览器开发者工具
   - 检查Console日志
   - 检查Network请求
   - 检查Application > Storage

3. **验证自动填充**
   - 截图数据应自动填充到分析表单
   - 图表信息应自动提取并显示

### 5. 调试技巧

#### 查看日志

1. **后台脚本日志**
   - 打开 `chrome://extensions/`
   - 找到Candlebot扩展
   - 点击"服务工作者"链接

   - 右键点击扩展图标
   - 选择"检查弹窗"
   - 查看Console和Network

#### 检查存储

   - 检查postMessage通信
   - 检查storage变化

### 6. 常见问题解决

#### 问题：扩展图标不显示

**可能原因：**
1. 扩展未正确加载
2. manifest.json配置错误
3. 图标文件缺失或格式错误

**解决方法：**
1. 重新加载扩展
2. 检查控制台错误
3. 验证图标文件存在且为PNG格式

#### 问题：截图失败

**可能原因：**
1. 缺少 `activeTab` 权限
2. 当前页面不支持截图
3. Chrome限制

**解决方法：**
1. 检查manifest.json权限配置
2. 确认在支持的图表页面
3. 尝试刷新页面后重试

#### 问题：数据传递失败

**可能原因：**
1. Web应用未运行
2. 跨域限制
3. 数据太大

**解决方法：**
1. 启动Web应用服务器
2. 检查host_permissions配置
3. 查看传输方法推荐

#### 问题：弹窗不显示

**可能原因：**
1. popup文件路径错误
2. HTML/CSS/JS错误
3. 扩展冲突

**解决方法：**
1. 检查manifest.json中的default_popup路径
2. 查看弹窗开发者工具错误
3. 禁用其他扩展测试
