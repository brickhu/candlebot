Candlebot扩展数据传递测试

此页面用于测试扩展与web app之间的数据传递。请确保web app运行在 http://localhost:5173。

测试1: 模拟扩展数据传递

点击下方按钮模拟扩展传递图片数据到web app
数据将通过URL参数传递到 /new 页面

测试2: 检查数据接收

打开浏览器开发者工具（F12）
查看Console标签页的日志输出
检查以下函数是否正常工作：
// 在web app控制台检查 checkExtensionRedirect() // 应该返回类似： // { // fromExtension: true, // imageData: "iVBORw0KGgoAAAANSUhEUg...", // source: "url_params" // }

测试3: 完整流程测试

1. 安装并加载Candlebot扩展
2. 访问TradingView或aggr.trade图表页面
3. 点击扩展图标进行自动截图
4. 在弹窗中查看截图预览
5. 点击"立即分析"按钮
6. 验证新标签页是否正确加载截图
验证点：
  • 扩展图标点击后是否自动截图
  • 弹窗是否显示截图预览
  • "立即分析"按钮是否跳转到分析页面
  • 分析页面是否自动加载截图
  • 分析弹窗是否显示扩展来源的图片
