# Candlebot v2.0 快速测试指南

## 测试环境准备

### 1. 启动后端服务
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 构建前端扩展
```bash
cd extension
npm install
npm run build
```

### 3. 加载Chrome扩展
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `extension/dist` 目录

## 功能测试流程

### 测试1: 用户注册和登录
1. 打开 aggr.trade 或 TradingView
2. 点击Candlebot扩展图标
3. 点击"登录"按钮
4. 在Dashboard中注册新账户
5. 登录账户

**预期结果**:
- Popup显示用户状态和剩余次数
- 可以正常登录/登出

### 测试2: 图表分析
1. 在支持的交易平台页面
2. 点击Popup中的"开始分析"按钮
3. 等待分析完成

**预期结果**:
- 成功截图并压缩图片
- 调用AI API进行分析
- 显示分析结果和评级
- 用户配额减少

### 测试3: Dashboard功能
1. 点击Popup中的用户名称（带→箭头）
2. 打开Dashboard页面

**测试项目**:
- [ ] 首页统计信息显示
- [ ] 历史记录列表
- [ ] 报告详情查看
- [ ] 问答功能
- [ ] 用户设置修改
- [ ] 套餐页面浏览

### 测试4: 历史记录管理
1. 在Dashboard中点击"分析记录"
2. 查看历史分析列表
3. 点击某个记录查看详情
4. 测试问答功能

**预期结果**:
- 显示所有历史分析记录
- 可以查看完整报告
- 可以基于报告提问

### 测试5: 配额限制
1. 进行多次分析直到配额用完
2. 尝试超额分析

**预期结果**:
- 配额用尽时显示错误信息
- 不能进行新的分析

## API端点测试

使用 `backend/test_api.py` 进行API测试：

```bash
cd backend
python test_api.py
```

测试的API端点:
- [ ] `GET /health` - 健康检查

- [ ] `POST /auth/login` - 用户登录
- [ ] `GET /auth/me` - 获取用户信息
